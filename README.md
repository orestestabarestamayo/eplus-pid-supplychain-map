# EcoChain – Supply Chain & Descarbonização 🌱

Plataforma integrada ao PID E+ para mapeamento inteligente de cadeias produtivas brasileiras com foco em sustentabilidade e redução de emissões de CO₂.

O projeto foi desenvolvido durante o Hackathon E+ com o objetivo de resolver problemas reais da economia brasileira por meio da visibilidade da cadeia de suprimentos, análise logística e recomendação de fornecedores sustentáveis.

---

# 🚀 Visão Geral

O **EcoChain** conecta empresas fornecedoras, centros de distribuição e pontos de venda em uma plataforma interativa.

A solução adiciona uma camada de inteligência capaz de:

- calcular a pegada de carbono da cadeia produtiva;
- simular cenários de redução de CO₂;
- recomendar fornecedores mais sustentáveis;
- visualizar a cadeia logística em mapa interativo;
- apoiar decisões estratégicas voltadas à descarbonização.

O sistema utiliza dados logísticos e simulações para promover transparência e apoiar a transição para uma economia mais verde.

---

# 🌍 Funcionalidades

## 📌 Mapa Interativo
Visualização geográfica da cadeia produtiva utilizando mapas interativos com fornecedores e pontos logísticos.

## 📊 Dashboard de CO₂
Painel inteligente com:

- emissão atual da cadeia;
- emissão simulada;
- economia de CO₂;
- percentual de redução.

## 🎚️ Simulação Interativa
Slider dinâmico para testar cenários sustentáveis em tempo real.

## 🟢 Recomendação Inteligente
Sugestão automática do fornecedor com menor impacto ambiental.

# 📄 Documentação da Solução

## Qual é o tipo de solução criada?
Plataforma web inteligente de mapeamento de cadeia produtiva com simulação de CO₂ e recomendação sustentável.

## Qual é o manual da solução?
O usuário acessa a plataforma via navegador, visualiza o mapa interativo da cadeia produtiva e utiliza o dashboard para simular cenários sustentáveis e analisar emissões de CO₂.

A navegação permite:
- visualizar fornecedores;
- analisar impacto ambiental;
- simular redução de carbono;
- identificar fornecedores sustentáveis.

## Qual é a viabilidade da solução?
A solução possui alta viabilidade técnica devido ao uso de tecnologias modernas, gratuitas e escaláveis.

A implementação pode ser realizada com infraestrutura cloud de baixo custo e equipes reduzidas de desenvolvimento.

Além disso, o projeto acompanha tendências globais de ESG, sustentabilidade e descarbonização industrial.

## Há soluções similares no mercado?
Sim. Existem plataformas de supply chain e monitoramento logístico, porém poucas integram visualização geográfica com simulação sustentável em tempo real.

O EcoChain foi inspirado em conceitos modernos de Supply Chain Intelligence e ESG Analytics.

## Quais são os diferenciais?
- Simulação interativa de CO₂ em tempo real;
- Recomendação inteligente de fornecedores;
- Visualização geográfica da cadeia produtiva;
- Interface intuitiva e acessível;
- Arquitetura moderna e escalável;
- Foco em ESG e descarbonização.

## Quais os possíveis impactos?
O EcoChain pode auxiliar empresas a:

- reduzir emissões de carbono;
- otimizar decisões logísticas;
- aumentar transparência da cadeia produtiva;
- fortalecer estratégias ESG;
- apoiar metas de sustentabilidade.

A solução também contribui para uma economia mais verde e inteligente.

# 🏗️ Arquitetura do Sistema

O EcoChain utiliza uma arquitetura híbrida e desacoplada.

## Frontend
Interface desenvolvida em React e Wix para proporcionar uma experiência moderna, rápida e responsiva.

## Integração de Dados
Uso de requisições assíncronas (`fetch`) para comunicação entre frontend e backend.

## Backend API
API construída em Node.js + Express responsável pela lógica de negócios, processamento de dados e simulações de CO₂.

## DevOps
Fluxo de Continuous Deployment (CD) integrado ao GitHub.

## Segurança
Configuração de CORS para comunicação segura entre frontend e backend.

# ⚙️ Como Executar o Projeto

## Clone o repositório

```bash
git clone https://github.com/orestestabarestamayo/ecochain-supply-map.git
```

## Acesse a pasta do projeto

```bash
cd ecochain-supply-map
```

## Instale as dependências

```bash
npm install
```

## Execute o projeto

```bash
npm run dev
```

O sistema estará disponível em:

```bash
http://localhost:5173
```
